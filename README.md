# Line-following-warehouse-bot

## PROJECT OVERVIEW
This project aims to automate and streamline warehouse inventory management by replacing manual barcode scanning with an autonomous robotic system. The robot navigates a predefined path, detects pallets or boxes in its trajectory, halts to scan the attached RFID tags, and wirelessly transmits the asset data to a centralized database in real-time. This reduces human error, increases auditing speed, and provides a scalable foundation for modern smart warehouses.

##Features
Line following navigation
Inventory management
Autonomous robot movement
